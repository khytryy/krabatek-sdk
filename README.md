# KrabaTekSDK - Software development kit for KrabaTek's CPUs

> [!WARNING]
> This project is under construction, so expect unimplemented stuff

KrabaTekSDK is a development kit featuring:

- Krabascript compiler (kt-XX-ks)
- C compiler (kt-XX-cc)
- Assembler (kt-XX-as)
- Linker (kt-XX-ld)
- Object copy (kt-XX-objcopy)
